The Blood Bank Record Keeper is a database-driven application designed to manage and maintain blood bank records efficiently. The system stores information about blood donors, blood recipients, blood groups, blood stock availability, donation history, and blood requests. It helps blood bank staff track blood inventory in real time, update donor records, and generate reports for better decision-making.

The project aims to reduce manual paperwork, improve data accuracy, and ensure quick access to critical information during emergencies. Features include donor registration, blood stock management, blood request processing, search and filtering of records, and report generation. The system enhances operational efficiency and supports timely blood distribution to patients in need.

Key Features
Donor registration and management
Blood group inventory tracking
Blood donation and request records
Blood stock availability monitoring
Search and update donor/recipient details
Report generation and data management
Secure and user-friendly interface


Technologies Used (example)
Frontend: HTML, CSS, JavaScript
Backend: Java
Database: MySQL


Objective
To automate blood bank record management, maintain accurate blood inventory, and facilitate quick retrieval of information for effective blood donation and distribution services.
